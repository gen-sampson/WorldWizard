API Link: https://restcountries.com/v2/all 

App Description: Users can view information about any country, as well as view
and boost a country's "coolness score". Countries start out with no score,
but anyone that uses the webapp may boost that country's score. Scores
are boosted in increments of 5. Score data is stored in MongoDB.
